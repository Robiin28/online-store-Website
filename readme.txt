how to configure the project

1. copy the folder called onlineshop to xampp inside the htdocs folder(xampp or wampp must be installed)
2. open xamp control pannel and start mysql and apache
3. go to phpMyAdmin, click on create database and create a database called onlinestore
4. click on import at the top of the page and input the onlinestore.sql within the db folder. then import the database
5. open your browser and type localhost and the project should appear
